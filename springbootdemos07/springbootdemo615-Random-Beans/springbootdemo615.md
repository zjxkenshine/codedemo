# 简介
java.util.RandomAPI提供了7种方法来生成随机数据：
```
nextInt()
nextLong()
nextDouble()
nextFloat()
nextBytes()
nextBoolean()
nextGaussian()
```
随机String与其他对象：`nextObject(Class type)`
