# 简介
Proxetta提供三种类型代理：
- Proxy：创建代理
- Wrapper：创建包装
- InvokeReplacer：替换方法调用

# Pointcut
有多种方式定义切面点，示例是使用自定义注解@Log

# Advice
用于访问目标方法

# ProxyProxetta
创建代理

# Paramo
获取方法形参名称
